Student Learning Notebook
Student Learning Notebook

The Student Learning Profile is a project that I worked on during my co-op placement at the Peel District School Board.

The application is used by teachers to capture and track thier student's progress and learning over a period of time.

The technologies I've implemented on this projects are as followed.

Backend: JAVA SPRING MVC, SQL ORACLE.

Upnote
Upnote

Upnote is a project I tinkered around during my free time.

I really wanted to explore the world of Ruby on Rails and decided to make an application out of this experience.

It is a bullet-driven journal or diary, that keeps track and organizes your daily activity using bullets.

There are different bullet types for the type of activity you are tracking. E.g. a task will be a checkbox, an idea will be a light-bulb icon...etc.

The technologies I've utilize in this project are Ruby on Rails, Google Polymer, Jquery, and MySQL.

Orbit
Orbit

The goal of the game is to absorb other planets to complete each stage.

If you are bigger than the other planets, your gravitational pull will drag them towards you, however if your are smaller than them, it works the opposite way!

Orbit is a game 2D gravity physics game built using the UNITY 3D engine and coded C#.

Growing up playing alot of MUDs in the 1990s, I was inspired to create a short text-based game called "Contagion".

Text-based games usually start with the character in some room and they have to find thier way through navigating to different rooms and interacting with objects by text commands.

Contagion was written in C.
